# My AI coding agents configs

This repository contains my personal configurations and scripts for AI coding agents. It includes skills, subagents, settings, tools, and resources that I use to enhance the performance and capabilities of AI coding assistants.

## Usage

Symlink skills to your global agent directory:

```bash
ln -s /Users/andrew/Projects/agents/skills ~/.claude/skills
```


Symlink subagents to your global agent directories for each agent:

```bash
ln -s /Users/andrew/Projects/agents/agents/copilot ~/.copilot/agents
ln -s /Users/andrew/Projects/agents/agents/claude ~/.claude/agents
```
